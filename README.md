# Password Generator (CLI)

## Description
A Python command-line application that generates strong random passwords.

## Features
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Custom password length

## How to Run
1. Open terminal.
2. Run:
   python Password.py
3. Enter the password length.
4. Get a strong random password.

## Expected Output

===== PASSWORD GENERATOR =====
Enter password length: 12

Generated Password:
A#7kP!2mQ@9x
